# Vectograph

The Vectograph is an audio modifier in the sense that it controls what the audio volume is at; it tells you what is playing (Via Spotify) and can pause/play the audio output, and can fast forward and rewind on a track (Via Spotify) using the keys, and can display audio output via the 10 LEDs and the keys or the rotary encoder can also adjust audio.
